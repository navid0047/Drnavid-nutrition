Dr.Navid Nutrition - PWA package for GitHub Pages
Files included:
- index.html
- manifest.json
- service-worker.js
How to publish on GitHub Pages:
1. Create a public GitHub repository (e.g. drnavid-nutrition).
2. Upload all files in this package to the repository (Add file -> Upload files) and Commit.
3. In repository Settings -> Pages -> Source: main branch / root -> Save.
4. After a minute, your site will be available at https://<username>.github.io/<repo>/ - open that URL in Safari and use Share -> Add to Home Screen.
